#{Introduction}
A small page layout displaying my personal profile, skills and career details.

##{conributors}
*Eldad Mwangi

###{Projects Purpos}
The display contains personal profile information on  education background and career information.

####{technology}
The project incorpates the use of HTML, CSS and a Bootstrap link for the font and icons on the menu-bar
The menu bar consists of a sub-menu with a dropdown.
On the body section, the slides move with the help of animation property and the slider property of CSS after the set duration.
 
 ##{Known bugs}
 The dropdwown menu on the menubar might not be responsive.
